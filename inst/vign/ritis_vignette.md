ritis introduction
==================

An interface to the Integrated Taxonomic Information System (ITIS)

## Setup

Install from CRAN


```r
install.packages("ritis")
```

Or install the development version from GitHub


```r
devtools::install_github("ropensci/ritis")
```

Load `ritis`


```r
library("ritis")
```
